## Style Guidelines for All Images

### Color Scheme
- Primary: Deep blue (#092433)
- Accent: Light blue, cyan
- Secondary: Green (#0ab476) for status indicators
- Neutral: White, professional grays
- Alert: Red (#eb4247) for warnings

### Lighting
- Professional, clean, well-lit environments
- Blue accent lighting for tech atmosphere
- Avoid harsh shadows
- Golden hour for exterior shots
- Cool lighting for interiors

### Quality Requirements
- High resolution, photorealistic where applicable
- Sharp focus on main subjects
- Professional depth of field
- Clean, noise-free images

### Composition
- Clean, organized, corporate aesthetic
- Professional framing
- Avoid cluttered backgrounds
- Show technical sophistication
- Emphasize scale and professionalism

### Avoid
- Cluttered environments
- Consumer-grade equipment
- Unprofessional setups
- Outdated technology
- Poor cable management

## Technical Specifications Summary

| Image Type | Aspect Ratio | Usage | Dimensions (min) |
|------------|-------------|-------|------------------|
| Hero backgrounds | 16:9 | Full-width headers | 1920×1080px |
| Landscape photos | 3:2 | Main content images | 1800×1200px |
| Portrait photos | 2:3 | Vertical blocks | 1200×1800px |
| Square icons/graphics | 1:1 | Cards, services, icons | 1200×1200px |

### File Format
- JPEG for photographs (quality 85-95%)
- PNG for graphics with transparency
- WebP for optimal web performance

### Color Profile
- sRGB for web use
- Color managed workflow

### Optimization
- Compress for web without quality loss
- Use responsive images
- Consider lazy loading for below-fold images
